# Grand Strategy Game Demo

一个融合《欧陆风云》和《维多利亚》玩法的大战略游戏demo。
该游戏demo旨在展示大战略游戏的核心玩法机制和系统设计。
该游戏demo将包含以下主要系统：

## 地图设计
该模块的意图为设计一个欧陆风云4风格的虚拟世界地图，包含国家边界、地形类型等信息。
具体设计要求见 /game/dev_md/map_design.md

## 资源产出
该模块的意图为设计游戏中的资源产出系统，包括原油、矿产等资源的产出点与产量
具体设计要求见 /game/dev_md/resource_production.md

## 市场系统
该模块的意图为设计游戏中的市场系统，模拟自发的市场商品定价，模拟根据不同的国家政策法规的不同程度自发的市场行为
具体设计要求见 /game/dev_md/market_system.md

## 工业系统
该模块的意图为设计游戏中的全面工业系统，包含工业产能、工业升级、工业原料的周转等
具体设计要求见 /game/dev_md/industrial_system.md

## 战争系统
该模块的意图为设计游戏中的战争系统，包含战争分数、陆军交战、海军交战等机制
具体设计要求见 /game/dev_md/war_system.md

## 科技树
该模块的意图为设计游戏中的科技树系统，包含科技研发、科技效果等机制
具体设计要求见 /game/dev_md/tech_tree.md

## 人口系统
该模块的意图为设计游戏中的人口系统，包含人口增长、人口迁移、人口结构等机制
具体设计要求见 /game/dev_md/population_system.md

## 事件系统
该模块的意图为设计游戏中的事件系统，包含随机事件、历史事件等机制
具体设计要求见 /game/dev_md/event_system.md

## 外交系统
该模块的意图为设计游戏中的外交系统，包含国家关系、联盟体系、外交谈判等机制
具体设计要求见 /game/dev_md/diplomacy_system.md

## 内政系统
该模块的意图为设计游戏中的内政系统，包含政策制定、社会稳定、经济管理、宫廷运作等机制
具体设计要求见 /game/dev_md/internal_affairs_system.md
