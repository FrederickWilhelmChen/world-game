# world-game
我需要你开发这样一款世界模拟游戏

## 前端：
- 构建一个墨卡托投影的世界地图，在世界地图上显示各个国家和地区的边界。精确度不需要很高，也不需要地形图，但要展示每个国家的轮廓。
- 世界地图上需要显示国家名称和首都名称。
- 当鼠标悬停在某个国家时，显示该国的数据信息
- 每个国家的数据需要网络爬虫拉取实时数据
- 数据信息包括以下内容
    - 1. 原油产量（桶/日）
    - 2. 粮食产量（分类别，吨/年）
    - 3. 有色金属产量（分类别，吨/年）
    - 4. 黄金产量（公斤/年）
    - 5. GDP（美元）

## 后端：
- 使用Python的Flask框架搭建后端服务器
- 实现一个API接口，当前端请求某个国家的数据时，返回该国的数据信息
- 使用网络爬虫技术从公开数据源获取各国的实时数据，一小时拉取一次，存储在该项目/data路径下的txt文件中，使用json格式存储数据
